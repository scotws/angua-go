The Executive Officer (xo) for Angua 
Scot W. Stevenson <scot.stevenson@gmail.com>
First version: 07. Nov 2018
This version: 07. Nov 2018

The xo receives the commands from the Command Line Interface (CLI) in angua.
It runs as a goroutine. It hides the messy details of the mode switching in
the CPU from the user. The xo knows about cpu8 and cpu16 
but not about mem.
